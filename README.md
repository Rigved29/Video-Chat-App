
# Video Chat App

It is a web-app for video calling. Every user in it has a uniqueID, with the help of it you can make a video call. It also has video, audio turn On/Off functionlity.

- [@Rigved29](https://github.com/Rigved29)

## Features

- Video calling using unique ID
- screen switching on picking up the call
- video and audio turn on/off
- Cross platform


## Run Locally

Clone the project

```bash
  git clone https://github.com/Rigved29/Video-Chat-App.git
```

Go to the project directory

```bash
  cd Video-Chat-App
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```


## Tech Stack

**Client:** React, socket.io, peerjs

**Server:** Node, Express, socket.io
